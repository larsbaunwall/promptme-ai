# PromptMe AI

A browser-based teleprompter application with advanced features for content creators and presenters.

## Features

- **Editable Text Box**: Type or paste your script directly in the browser
- **Custom Emphasis**: Use `[square brackets]` to highlight important text in red
- **Adjustable Font Size**: Change text size from 16px to 120px
- **Variable Speed Control**: Adjust scroll speed from 1 to 10
- **Play/Pause Control**: Start and stop scrolling with a single button
- **Vertical Mirroring**: Flip text vertically for teleprompter glass setups
- **Horizontal Mirroring**: Flip text horizontally for reverse reading
- **No Installation Required**: Runs entirely in the browser with no dependencies

## Usage

Simply open `index.html` in any modern web browser. No build process or installation required.

### Controls

- **Play/Pause Button**: Toggle text scrolling
- **Speed Slider**: Adjust scroll speed (1-10)
- **Font Size Input**: Set text size in pixels
- **Vertical Mirror Button**: Flip text upside down
- **Horizontal Mirror Button**: Flip text left to right

### Text Emphasis

Wrap text in square brackets to make it appear in red and bold:

```
This is normal text.
This is [emphasized text] in red.
```

## Technology

Built with vanilla JavaScript and modern CSS - no frameworks or build tools required for maximum simplicity and compatibility.
